# IAM Policy Templates

Identity and Access Management policy templates and checklists for fintech, enterprise, and digital products.

Built by TrustGrid Technology Limited.

## Templates Included

| Template | Description |
|---|---|
| Access Control Policy | Organisation-wide policy for managing user access |
| User Provisioning and Deprovisioning | Process for onboarding and offboarding user access |
| Privileged Access Management | Controls for managing admin and privileged accounts |
| Password Policy | Standards for password creation and management |
| MFA Implementation Checklist | Steps for rolling out multi-factor authentication |
| Access Review Template | Quarterly access review process and sign-off sheet |

## Who This Is For

Fintech companies, digital lending platforms, healthcare organisations, SaaS products, and any business managing user access to sensitive systems.

## How to Use

1. Select the template relevant to your needs
2. Customise it to reflect your organisation name and systems
3. Get sign-off from relevant stakeholders
4. Implement and communicate to your team
5. Review annually or after any significant access incident
